# Code-Cards
A new card type for Anki Decks.


## For Software Engineers
I created this card style for my 100Devs crew. It's a three field cardset that allows the user to put in a coding question, then has a code box, then will respond with whether the solution is correct or not. If you give up, you can always check the answer, and the correct answer will be put up. I've worked hard to correct issues related to spacing and line-breaks. This add-on will take note of how many spaces you use as well as line breaks.
