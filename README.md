# Code-Cards
A new card type for Anki Decks.


## For Software Engineers
I created this card style for my 100Devs crew. It's a three field cardset that allows the user to put in a coding question, then has a code box, then will respond with whether the solution is correct or not. If you give up, you can always check the answer, and the correct answer will be put up. I've worked hard to correct issues related to spacing and line-breaks. This add-on will take note of how many spaces you use as well as line breaks.

## Steps to Add to Deck:
1. Go to Tools > Manage Note Type > Add > Add:Basic
2. Name it: Coding Card
3. Select **Fields** and ensure you have at least three fields:
   a. Front (The question/prompt)
   b. CodeTemplate (The boilerplate code with blanks, if any) - this should be in number 2 slot (you might have to rearrange
   c. Back (The correct solution code)
4. Make sure you have **Coding Card** highlighted and then select **Cards** to edit the template.
5. Paste in the files located in this repository in their correct positions. <br/>
    a. frontTemplate goes in Front Template. <br/>
    b. backTemplate goes in Back Template. <br/>
    c. Styling goes under Styling.
6. Hit **Save**
7. Now, when you want to make a coding question, just make sure your card type is set to Coding Card, and you should be all set! See example of look below.
